# 🪨 Stone Paper Scissors Game

A professional CLI-based Stone Paper Scissors game built with Python.

## 🚀 Features
- **Modular Code:** Logic and UI are separated.
- **Input Validation:** Handles case-sensitivity (stone/STONE/Stone).
- **Random AI:** Computer uses `random.choice` for fair play.
- **Color Coded:** Interactive terminal experience with ANSI colors.

## 🛠️ How to Run
```bash
python main.py
